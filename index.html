História da Web

A World Wide Web, a Rede de alcance mundial em português ("WWW" ou simplesmente "Web") é um meio de comunicação global no qual utilizadores podem ler e escrever através de computadores conectados à Internet. O termo Web é usado erradamente como sinónimo da própria Internet, sendo a Web apenas um serviço que utiliza a Internet, assim como as mensagens de e-mail; a História da Internet antecede bastante a da Rede de alcance mundial.

A parte que tem hipertexto da Web tem uma história complicada; notáveis influências e precursores incluem o Memex de Vannevar Bush,[1] a Standard Generalized Markup Language (SGML) da IBM e o Projeto Xanadu de Ted Nelson.

O conceito de um sistema de informações global/doméstico é tão antigo quanto en:A Logic Named Joe, uma pequena história de 1946 criada por Murray Leinster. Nela, todas as casas possuíam terminais de computador, chamados logics. Embora fosse um sistema centralizado de forma autônoma a história reflete o sentimento de ubiquidade da informação, que veio com a Web.

" Em Agosto de 1984 escrevi um artigo ao Chefe do Grupo SW (do CERN) Les Robertson, para descrever um projecto piloto a fim de instalar e avaliar o protocolo TCP/IP em algumas máquinas não Unix do CERN [...] Cerca de 1990 o CERN tinha se tornado o maior sítio da Internet da Europa [...] e do mundo. Um resultado chave de todos estes factos foi que cerca de 1989 a rede internet do CERN estava a tornar-se a medida a partir da qual Tim Berners-Lee viria a criar a World Wide Web como uma ideia verdadeiramente ideal... " Ben Segal, Short History of Internet Protocols at CERN, April 1995

O que são Clients

(voltar)

Um cliente é um programa que, como parte de sua operação, depende do envio de uma solicitação a outro programa ou a um hardware ou software de computador que acessa um serviço disponibilizado por um servidor (que pode ou não estar localizado em outro computador). Por exemplo, navegadores da web são clientes que se conectam a servidores da web e recuperam páginas da web para exibição. Clientes de e-mail recuperam e-mails de servidores de e-mail.

O que são Servers

(voltar)

Um servidor é um computador que fornece informações a outros computadores chamados " clientes " em uma rede de computadores. Essa arquitetura é chamada de modelo cliente-servidor. Os servidores podem fornecer várias funcionalidades, geralmente chamadas de "serviços", como compartilhar dados ou recursos entre vários clientes ou executar cálculos para um cliente. Um único servidor pode atender a vários clientes e um único cliente pode usar vários servidores. Um processo cliente pode ser executado no mesmo dispositivo ou pode se conectar por meio de uma rede a um servidor em um dispositivo diferente. Os servidores típicos são servidores de banco de dados, servidores de arquivos, servidores de e-mail, servidores de impressão, servidores da web, servidores de jogos e servidores de aplicativos. Os sistemas cliente-servidor são geralmente implementados com mais frequência (e frequentemente identificados com) o modelo de solicitação-resposta : um cliente envia uma solicitação ao servidor.

Linguagens de Programação

(voltar)

Linguagens de programação são sistemas formais que permitem aos programadores criar instruções para computadores, definindo como eles devem realizar certas tarefas. Essas linguagens são compostas por um conjunto de regras sintáticas e semânticas que permitem expressar algoritmos e comandos de forma que a máquina possa interpretá-los e executá-los.

Basicamente, linguagens de programação são ferramentas que permitem a comunicação entre humanos e computadores. Elas fornecem uma maneira estruturada de escrever instruções que o computador pode entender e executar, resultando em software e sistemas.

Exemplos de linguagens de programação:

  1. Python:

    Conhecida pela sua simplicidade e legibilidade, amplamente usada em ciência de dados, inteligência artificial e desenvolvimento web.

  2. Java:

    Versátil e multiplataforma, usada em aplicações corporativas, Android e grandes sistemas.

  3. JavaScript:

    Essencial para desenvolvimento web, especialmente na criação de interações dinâmicas com o usuário.

  4. HTML:

    HTML (HyperText Markup Language) é a linguagem padrão para criar e estruturar páginas web. Ela usa tags para definir como o conteúdo (textos, imagens, links, etc.) é exibido em um navegador. Embora não seja uma linguagem de programação, o HTML é fundamental para a estrutura e organização de qualquer página na web.
